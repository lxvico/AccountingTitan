# AccountingTitan

A static WordPress site based on an exported ZIP file.

## Structure
- `wordpress-export/` – place the exported WordPress ZIP file here.
- `site/` – extracted static site files ready to be served.

## Getting started
1. Copy your exported WordPress ZIP into `wordpress-export/`.
2. Extract it into the `site/` directory.
3. Serve `site/` with any static file server or hosting provider.
